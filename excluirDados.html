Como deletar os dados de usuário?

Última atualização: 21/11/2025

1. Exclusão de Dados Pessoais

Os usuários do Memória Viva podem solicitar a exclusão completa de seus dados a qualquer momento. Para atender às políticas de privacidade e às diretrizes de plataformas como Google e Meta, disponibilizamos canais oficiais para remoção de informações.

1.1 Como solicitar a exclusão dos dados

Para excluir seus dados, envie uma solicitação para o e-mail abaixo com o assunto: “Solicitação de Exclusão de Dados – Memória Viva”.

Sua solicitação será processada e todos os dados associados à sua conta serão removidos dos nossos sistemas (incluindo informações no Firebase Auth, Firestore e Storage) dentro de até 7 dias úteis.

1.2 Informações necessárias no pedido

  • Nome completo
  • E-mail utilizado no app
  • Descrição breve confirmando que deseja excluir os dados

1.3 O que será excluído

Todos os dados armazenados pelo Memória Viva, incluindo:

  • Conta de autenticação
  • Fotos, vídeos e mensagens pessoais salvos no app
  • Dados de preferências, uso e histórico

1.4 Consequências da exclusão

Após o processo, não será possível recuperar nenhuma informação apagada, e o acesso ao aplicativo poderá ser desativado dependendo das funções utilizadas.
